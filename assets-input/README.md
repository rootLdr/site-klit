# assets-input/

Pasta oficial para você colocar arquivos **pelo Windows** (via `\\192.168.68.99\site-klit\assets-input`).

| Pasta | O que colocar aqui |
|-------|--------------------|
| `logos/` | Logo da KLIT e variações (claro/escuro), favicon, marcas das soluções |
| `images/` | Fotos, banners, imagens de hero e seções |
| `references/` | Sites/visuais de referência para calibrar a direção premium |
| `docs/` | PDFs, documentos institucionais, manual de marca, lista de clientes |

Regra: mantenha os assets **somente aqui**. O site lerá/copiará daqui — nada de espalhar arquivos em outras pastas sem decisão documentada.

Detalhes de acesso pelo Windows: ver `../docs/windows-access.md`.
